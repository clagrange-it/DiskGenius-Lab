# DiskGenius-Lab
Migration d'OS sur un autre disque dur

## Présentation

Ce projet présente la migration de Windows d'un disque mécanique sur un disque SSD.

## Technologies utilisées

- Windows 11
- DiskGenius

## Etapes

1. Installation du SSD dans le boîtier
2. Installation de DiskGenius sur le disque mécanique
3. Vérification de l'absence de données sur le SSD
4. Sélection du disque cible
5. Vérification finale avant migration
6. Migration en cours
7. Redémarrage système
8. Vérification de l'installation de Windows 11
9. Estinction du poste, débranchement du disque mécanique et redémarrage sur le SSD
10. Migration terminée, formatage du disque mécanique possible
